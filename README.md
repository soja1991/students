# students

## Getting started

Assuming you've already installed on your machine: PHP (>= 7.3.0), [Laravel](https://laravel.com), [Composer](https://getcomposer.org).

``` bash
# install dependencies
composer install

Then create table "student_management"

Then launch the server:

php artisan migrate

php artisan db:seed

``` bash
php artisan serve
```
